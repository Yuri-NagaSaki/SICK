catbash.net

Server scripts that just run.

猫脚本 (Catbash) is a small suite of open-source Linux tools for ops and lab machines. No installers, no daemons — curl | bash and you're done.

SICK · hardware NETS · iperf3 CPUX · Geekbench MIT · open source
Menu · catbash.sh ba.sh · catbash
$ curl -sL https://ba.sh/menu | bash
$ curl -sL https://catbash.net/menu | bash
$ curl -sL https://ba.sh/menu | bash -s -- nets -r
$ curl -sL https://ba.sh/menu | bash -s -- sick -cn
$ curl -sL https://ba.sh/menu | bash -s -- cpux all

Interactive: 1 SICK · 2 NETS · 3 CPUX · 0 exit. Or pass sick/nets/cpux after bash -s --.
