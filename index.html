Hardware without the sick.

One command. Full inventory. Colorized tables for CPU, RAM, SMART, NVMe, RAID, NICs and GPU — or JSON. No daemon. No footprint.

one-shot · zero footprint EN / 中文 JSON MIT
Quick start ba.sh/sick
$ curl -sL https://ba.sh/sick | bash
$ curl -sL https://ba.sh/sick | bash -s -- -cn

-cn Chinese · -y auto-install · --json machine output · bash -s -- …

Report coverage

SystemHostname, OS, kernel, uptime
CPU / PlatformCores, freq, temp, NUMA, virt
MemoryTotals + per-DIMM table
Disks / SMARTHealth, hours, I/O, defects
NVMeTemp, endurance, spare
RAIDmdraid + HBA controllers
NetworkLink, speed, errors
GPU / BoardClass, model, BIOS
