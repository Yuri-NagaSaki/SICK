Network Endpoint Throughput Sampler

Public iperf3 tests (IPv4 / IPv6) against a curated set of providers — Leaseweb, Clouvider, Eranium, OnlyServers, RedSwitches 25G, Advin, and more. One curl.

iperf3 send + recv IPv4 · IPv6 16 endpoints MIT
Quick startcatbash.net · ba.sh
$curl -sL https://catbash.net/nets | bash
$curl -sL https://ba.sh/nets | bash
$curl -sL https://catbash.net/nets | bash -s -- -r

-r reduced · --region na|eu|apac · -4/-6 · -t 10 · -P 4 · needs iperf3 + python3
