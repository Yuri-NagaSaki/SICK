NETS · 猫脚本

Network Endpoint Throughput Sampler

Public iperf3 tests (IPv4 / IPv6) against a curated set of providers — DataPacket, Fiberby, Kamel, BuyVM, Leaseweb, Clouvider, Eranium, and more. One curl.

iperf3 send + recv IPv4 · IPv6 20 endpoints MIT
Quick start ba.sh · catbash.net
$ curl -sL https://ba.sh/nets | bash
$ curl -sL https://catbash.net/nets | bash
$ curl -sL https://catbash.net/nets | bash -s -- -r

-r reduced · --region na|eu|apac · -4/-6 · -t 10 · -P 4 · needs iperf3 + python3
